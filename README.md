# Playlist CRUD

Aplicación web para gestionar una playlist colaborativa del curso, permitiendo a los usuarios agregar y reproducir canciones, así como marcar las canciones escuchadas durante las sesiones.

## 📋 Descripción

Esta aplicación permite a los estudiantes y profesores mantener una playlist colaborativa para el curso. Las características principales incluyen:

- Reproducir canciones mediante enlaces directos
- Añadir nuevas canciones a la playlist
- Marcar canciones como escuchadas durante el curso
- Persistencia de datos usando almacenamiento local (localStorage)

## 🚀 Tecnologías Utilizadas

- HTML5 para la estructura
- CSS3 para el diseño y estilo
- JavaScript (vanilla) para la funcionalidad y lógica
- LocalStorage para guardar los datos localmente

## 🔧 Instalación

Al ser una aplicación web con HTML, CSS y JavaScript puro, no requiere instalación especial:

1. Clona este repositorio o descarga los archivos
2. Abre el archivo `index.html` en tu navegador

## 📝 Uso

### Reproducir una canción
1. En la página principal, simplemente haz clic en el enlace de la canción que deseas escuchar
2. La canción se reproducirá directamente

### Añadir una nueva canción
1. Haz clic en el botón "Añadir canción" en la página principal
2. Se abrirá la página del formulario
3. Completa los campos requeridos (nombre, artista, enlace, etc.)
4. Haz clic en "Guardar" para añadir la canción a la playlist
5. Serás redirigido automáticamente a la página principal

### Marcar una canción como escuchada
1. En la página principal, encuentra la canción que has escuchado
2. Marca la casilla de verificación junto a ella
3. El estado se guardará automáticamente en localStorage y permanecerá guardado entre sesiones
